Context AR

This project was built by a team of 6 for Portland State University Computer Science Capstone Project

Context AR is a web application built for instructors and students.

With this web app, instructors can build learning modules to guide students through a city using a gps based tour system.

Each learning module is composed of individual stages. Each stage will have a map location, and descriptive steps or goal.


Home Map View

This view will show modules near your area.

Module List View

Here you can find a list of the available modules you have access to.

View of the first step in a module

Here you can see the buttons for traversing the module, open a google maps linke, and finding you active position.

Module Complete View

This is the view for completing a module.

To show instructors you have completed the module, you can use email, reddit, facebook and email.
